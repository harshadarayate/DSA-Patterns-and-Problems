Single Non-Duplicate Element

LeetCode #540 [Medium]

Problem Statement

You are given a sorted array nums where every element appears twice except for one element which appears only once. Find and return the single element that appears only once.

Example: nums = [1,1,2,3,3,4,4,8,8] â†’ Output: 2

Approach 1: Binary Search on Pairs

Explanation: Use binary search to find where the pattern of pairs breaks. Before the single element, pairs start at even indices (0, 2, 4...), and after it, pairs start at odd indices (1, 3, 5...).

Time Complexity: O(log n)

Space Complexity: O(1)

# nums is a sorted array where every number appears twice except one
function findSingleElement(nums):
    left = 0
    right = length(nums) - 1

    while left < right:
        mid = (left + right) / 2

        if mid is odd:
            mid = mid - 1

        if nums[mid] == nums[mid + 1]:
            left = mid + 2
        else:
            right = mid

    return nums[left]

Approach 2: XOR Method

Explanation: XOR cancels out duplicate numbers since a ^ a = 0. When all numbers are XORed together, the duplicates vanish and only the single number remains.

Time Complexity: O(n)

Space Complexity: O(1)

function findSingleElement(nums):
    result = 0
    for num in nums:
        result = result XOR num
    return result

Approach 3: Linear Scan (Naive)

Explanation: Scan through the array and check each number. The number that doesnâ€™t match its neighbor is the single one.

Time Complexity: O(n)

Space Complexity: O(1)

function findSingleElement(nums):
    n = length(nums)
    if n == 1:
        return nums[0]

    for i in range(0, n):
        if (i == 0 and nums[i] != nums[i + 1]) or
           (i == n - 1 and nums[i] != nums[i - 1]) or
           (nums[i] != nums[i - 1] and nums[i] != nums[i + 1]):
            return nums[i]
